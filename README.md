# To-Do List

## [Live Demo](https://sodot-backwards.herokuapp.com)

### Installation

`bundle install`

`npm install`

### Start Server

`rails server`

Go to: localhost:3000

### Technologies

* React.js
* Rails v5.2.3
* PostgreSQL

### Features

* Create TODO
* Delete TODO
* Update TODO
* All / Complete / Incomplete TODO Filters
* Delete all Completed Todo
